Purpose

üìä Component 2: BrainSignalsChart

Purpose

Visualizes cognitive focus patterns over time using an SVG-based line chart.

Visual Structure

  • Large current focus percentage (36px, light blue)
  • SVG line chart with smooth curves
  • Gradient area fill under the line
  • Grid lines at 0, 25, 50, 75, 100
  • Y-axis labels
  • Two stat cards: High-focus and Low-focus periods

Props

data: Object ‚Äî Contains focusTimeSeries array and liveTracking

Technical Details

  • Generates SVG path using quadratic curves for smoothness
  • Responsive viewBox (600x240)
  • 40px padding on all sides
  • Calculates high/low focus percentages (threshold: 75%)
  • Linear gradient definition for area fill

ü§ñ Component 4: AIRecommendations

üìù Component 5: AssignTaskModal

Purpose

Modal dialog for assigning recommended tasks to employees with scheduling.

Visual Structure

  • Dark modal overlay with backdrop blur
  • Centered modal card (max-width: 500px)
  • Task name (read-only)
  • Employee name (read-only)
  • Start date picker (required)
  • Start time picker (required)
  • Optional notes textarea
  • Cancel and Confirm buttons

Props

task: Object ‚Äî Task data from recommendation
employeeName: String ‚Äî Name to display
onClose: Function ‚Äî Close modal handler
onConfirm: Function(taskData) ‚Äî Submit handler

Form Validation

  • Date and time fields are required (HTML5 validation)
  • Form submit triggers onConfirm with merged data
  • Clicking overlay closes modal
  • StopPropagation on modal content prevents accidental closure
